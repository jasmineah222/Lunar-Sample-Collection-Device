# Lunar-Sample-Collection-Device

# Project Overview 
***CAD engineering design project inspired by NASA Micro-G Next challenges for the Artemis Program. This device is for quick data sampling of the lunar surface. An example where this product will play a role is the case where the mission is cut short and unfortunately, extensive testing cannot be performed.***

# Technical Specifications & Features
This project was a mechanical device made for quick data collection during lunar missions. There are no electrical elements, as this is intended to be a purely mechanical engineering project. Solidworks FEA simulations were conducted with different materials to determine the most feasible option. This repository is a showcase of a few results performed in detail to give a glimpse of the project's process. There are also additional snippets of the FEA simulation results for different measurements (displacement and stress).
